== Installation ==

To install, simple extract the 'olympic-preloader' folder into your 'wp-content/plugins/' directory. Once extracted, you must activate the plugin within the WordPress Site Admin 'Plugins' section. Then go to Widget section and you'll find out the olympic preloader plugin.
